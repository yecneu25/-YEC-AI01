# 🚀 YEC AI Chat - Trợ lý Kinh tế học (NVIDIA NIM)

Giao diện chat AI chuyên nghiệp được thiết kế riêng cho **Young Economists Club (YEC)**, sử dụng sức mạnh của **NVIDIA NIM API** với các dòng model Llama 3.1 hàng đầu.

![Banner](assets/nvidia-logo.png)

## ✨ Tính năng nổi bật
- **⚡ Phản hồi siêu tốc:** Tận dụng hạ tầng của NVIDIA để trả lời câu hỏi trong tích tắc.
    - **Nemotron 70B:** Thông minh, phân tích sâu.
    - **Llama 3.1 8B:** Tối ưu tốc độ.
    - **Llama 3.1 405B:** "Quái vật" tri thức.
- **🛡️ Proxy Backend:** Bảo vệ API Key và giải quyết lỗi CORS triệt để.
- **🎨 Giao diện Premium:** Dark Mode tinh tế với tông màu cam đặc trưng của YEC.
- **📱 Responsive:** Hoạt động mượt mà trên cả máy tính và điện thoại.

## 🛠️ Hướng dẫn cài đặt

1. **Clone dự án:**
   ```bash
   git clone https://github.com/dangtphyec-crypto/-YEC-AI01.git
   ```

2. **Cấu hình API Key:**
   - Mở file `server.js`.
   - Tìm dòng `NVIDIA_API_KEY` và thay bằng Key của bạn (hoặc thiết lập biến môi trường).

3. **Chạy Server:**
   ```bash
   node server.js
   ```

4. **Truy cập:**
   Mở trình duyệt và truy cập `http://localhost:3000/`.
